# ml-practice-projects
Machine Learning Practice Projects | Hands-on implementations of clustering, classification, and regression algorithms using Python and Scikit-learn.
This repository contains a collection of Python-based machine learning experiments and datasets, ranging from data preprocessing to advanced classification and regression models.

## 📂 Repository Structure

### 🧪 Experiment Scripts
* **Regression:** Linear and Polynomial regression models (e.g., `Salary_dataset.py`, `Exp2_2.py`).
* **Classification:** Implementations of SVM, K-Nearest Neighbors, and Decision Trees (e.g., `Exp3_1Svm.py`, `Exp4-1.py`).
* **Clustering:** Unsupervised learning techniques like K-Means (e.g., `26Feb2026_Exp4_2.py`).
* **Lab Records:** Daily experiment logs organized by date (e.g., `Jan2026` to `Mar2026` logs).

### 📊 Datasets
The following datasets are used across various experiments:
* `AirQuality.csv` - Environmental data for regression.
* `breast-cancer.csv` - Binary classification tasks.
* `Telco_Customer_Churn.csv` - Customer behavior analysis.
* `iris.csv` & `house_prices.csv` - Standard ML benchmarking datasets.

## 🚀 Getting Started
1. Clone the repo: `git clone https://github.com/KhusheeVashisht/ml-practice-projects.git`
2. Install dependencies: `pip install pandas scikit-learn matplotlib`
3. Run any experiment: `python 12MAR_PracticalExam.py`
